    vuePress-theme-reco Pika    2015 - 2024

    Pika 是一个以 RocksDB 为存储引擎的的大容量、高性能、多租户、数据可持久化的弹性 KV 数据存储系统。Pika 力求在完全兼容 Redis 协议、 继承 Redis 便捷运维设计的前提下， 通过持久化存储的方式解决 Redis 在大容量场景下的问题，如单线程易阻塞、服务器资源有限、加载数据慢、故障切换成本高。

    快速上手 →

    平滑迁移

    完全兼容 Redis 协议，不用修改代码即可平滑从 Redis 迁移到 Pika

    完整支持

    支持大部分 Redis 中常用的数据结构

    冷热数据

    对热数据做缓存，将全量数据持久化存储到 RocksDB，并且实现冷热分级存储

    极大容量

    支持百 GB 的数据量级，能极大减少服务器资源占用和增强数据的可靠性

    轻松部署

    支持单机主从模式(slaveof)和 Codis 集群模式，能够轻易实现扩缩容

    便于运维

    提供了完整的运维命令和详细文档，能够轻松地上手运维
